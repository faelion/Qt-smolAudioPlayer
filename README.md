# Qt smolAudioPlayer
 A simple project to learn Qt Creator!
 Version: 6.6.1
 Desktop_x86_windows_msvc2019

 This is a one day project just to test the knowledge I can get using this framework in for the first day.

 ## Features
 - Drag and drop mp3 files
 - Audio name display
 - Play and pause
 - Volume Slider
